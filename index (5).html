Connect your Claude API key

Draftline calls the Claude API directly from your browser to run onboarding and build your site. Your key is stored only in this browser's local storage — it is never sent anywhere except Anthropic's API.

Because this runs client-side, anyone who opens this page's dev tools can see the key while it's active. Don't share this page publicly with your key attached — for a public/shared deployment, route requests through your own backend proxy instead.
